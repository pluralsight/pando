## Import

```javascript
import { useSubmenuInteraction } from '@pluralsight/react-utils'
```

```typescript
useSubmenuInteraction()
```

## Reference

Call `useSubmenuInteraction` at the top level of the Submenu component to which you would like to add interactive features:

```jsx showLineNumbers {2,6,8}
export function Submenu(props) {
  const submenuInteractionProps = useSubmenuInteraction()

  return (
    <li>
      <button {...submenuInteractionProps.trigger}>{props.label}</button>

      <menu {...submenuInteractionProps.menu}>{props.children}</menu>
    </li>
  )
}
```

### Parameters

This hook doesn't accept any parameters.

### Returns

An Object containing an `expanded` property of a Boolean value related to the state of the menu. A `menu` property to containing both event delegation and accessbility properties to spread on the `menu` element. A `trigger` property containing event delegation, ref, and dataset properties to spread on the element responsible for triggering the state of the menu (usually a `button`).
