```typescript
  disabled?: boolean
  sentiment?: 'default' | 'action' | 'danger'
  usage?: 'filled' | 'outline' | 'text'
  size?: 'm' | 'l'
  icon?: 'start' | 'end'
```

2. Label

## Usage

### Basic Buttons

```jsx live
function BasicButtons() {
  const gridItem = getGridItemProps({ colSpan: 2 })

  return (
    <div {...getGridProps({ cols: 12, gap: 8 })}>
      <div {...gridItem}>
        <button {...getButtonProps().button}>Action</button>
      </div>

      <div {...gridItem}>
        <button {...getButtonProps({ sentiment: 'default' }).button}>
          Default
        </button>
      </div>

      <div {...gridItem}>
        <button {...getButtonProps({ sentiment: 'danger' }).button}>
          Danger
        </button>
      </div>

      <div {...gridItem}>
        <button {...getButtonProps({ usage: 'outline' }).button}>
          Outline
        </button>
      </div>

      <div {...gridItem}>
        <button {...getButtonProps({ usage: 'text' }).button}>Text</button>
      </div>

      <div {...gridItem}>
        <button {...getButtonProps({ disabled: true }).button}>Disabled</button>
      </div>
    </div>
  )
}
```

### Button with icon

```jsx live
function ButtonWithIcon() {
  const gridItem = getGridItemProps({ colSpan: 3 })
  const { button, iconOptions } = getButtonProps({
    icon: 'start',
  })
  const { button: trailButton } = getButtonProps({
    icon: 'end',
  })

  return (
    <div {...getGridProps({ cols: 12, gap: 16 })}>
      <div {...gridItem}>
        <button {...button}>
          <PlaceholderIcon {...getIconProps(iconOptions)} />
          Leading icon
        </button>
      </div>

      <div {...gridItem}>
        <button {...trailButton}>
          Trailing icon
          <PlaceholderIcon {...getIconProps(iconOptions)} />
        </button>
      </div>
    </div>
  )
}
```

### Sizes

```jsx live
function ButtonSizes() {
```

## Reference

```typescript
getButtonProps(options?: ButtonOptions)
```

### Parameters

- `options`: A String value of a theme sentiment `'default' | 'action' | 'danger'` which determines the styling color theme produced. An optional usage property that accepts a String value of `'filled' | 'outline' | 'text'` to determine the styling. An optional size property that accepts a String value of either `'m' | 'l'` to display the size styles. An Optional disabled property of a Boolean value to display the disabled styles and accessbility properties. An optional icon property that accepts a string value of `'start' | 'end'` to display styling support for icons within a Button.

### Returns

An Object matching the [Anatomy](#anatomy) containing `className` and a11y properties relevant to each element/section. Additionally, it returns a `iconOptions` Object to use as the parameters for the Icon API.

## Behavior

### Patterns

**Do** use the Button for an actionable event needed from the user.

**Don't** use the Button for navigation or styled links. Instead use the TextLink component.

**Do** keep Button usage minimal within an area.

**Don't** use more than three Buttons in a single area.

**Do** space Buttons apart by 16px (1rem).

**Don't** space Buttons apart greater or less than 16px (1rem).

### Sentiment

**Do** use the "action" sentiment for all use cases where possible.

**Don't** use the "default" sentiment _unless_ it is a **last resort**.

**Do** use the "danger" sentiment for cancel or destructive actions.

**Don't** use the "default" sentiment for cancel or destructive actions.

### Usage

**Do** use the "outline" usage for a **lower priority alternative** use case to the "default" sentiment.

**Don't** use the "outline" usage for cancel or destructive actions.

**Do** use the "text" usage for the **lowest** priority action.

**Don't** use the "text" usage as a high-priority action.

### Sizing

**Do** use the default size for all use cases where possible of the Button.

**Don't** use the "m" size _unless_ it is a **last resort**.

## Accessbility

The Pando Button, and Icon APIs combined with semantic HTML allow the Button to be fully accessible and screen-readable.
