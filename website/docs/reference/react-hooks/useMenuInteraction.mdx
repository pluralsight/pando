<p className="size-xxl">A hook for handling menu interactions and state.</p>

:::note

This hook is for a top-level pop-up [menu](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/menu_role) and **not** a [menubar](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/menubar_role) role or submenus within either.
For submenus, see the [useSubmenuInteraction](./useSubmenuInteraction.mdx) hook.

:::

## Import

```javascript
import { useMenuInteraction } from '@pluralsight/react-utils'
```

```typescript
useMenuInteraction()
```

## Reference

Call `useMenuInteraction` at the top level of the Menu component you would like to add interactive features to:

```jsx showLineNumbers {2,8,11-12}
export function Menu(props) {
  const menuInteractionProps = useMenuInteraction()

  return (
    <div>
      <button
        {...menuStyles.trigger}
        {...menuInteractionProps.trigger}
      >
      ...
      {menuInteractionProps.expanded &&
        <menu {...menuProps.menu} {...menuInteractionProps.menu}>
          {props.children}
        </menu>
      }
```

### Parameters

This hook doesn't accept any parameters.

### Returns

An Object containing a `expanded` property of a Boolean value related to the state of the menu. A `menu` property to containing both event delegation and accessbility properties to spread on the `menu` element. A `trigger` property containing event delegation, ref, and dataset properties to spread on the element responsible for triggering the state of the menu (usually a `button`).
