
```typescript
import type {
```

## Usage

### Basic 2 Column Layout

```jsx live
function TwoColLayout() {
  function getGridItemStyles(options) {
    const styles = getGridItemProps(options)
    return {
      style: {
        ...styles.style,
        backgroundColor: 'var(--ps-surface-strong)',
        borderRadius: '6px',
        height: '11rem',
        marginBottom: '1rem',
        padding: '1rem',
      },
    }
  }

  return (
    <div {...getGridProps({ cols: 12, gap: 32 })}>
      <div {...getGridItemStyles({ colSpan: 9 })}>main section</div>
      <div {...getGridItemStyles({ colSpan: 3 })}>sidebar</div>
    </div>
  )
}
```

## Reference

### `getGridProps`

```typescript
getGridProps(options?: GridOptions): ReturnProps
```

#### Parameters

```typescript
interface GridOptions {
  classNames?: string[]
  cols?: number
  gap?: 6 | 8 | 12 | 16 | 32
  rows?: number
}
```

#### Returns

1. An Object matching the [Anatomy](#anatomy) containing `className` and a11y properties relevant to the label element including an unused class consisting of the naming pattern `pando-grid`.
2. A `style` property that adds dynamic styling based on the options provided.

### `getGridItemProps`

```typescript
getGridItemProps(options?: GridItemOptions): ReturnProps
```

#### Parameters

```typescript
interface GridItemOptions {
  classNames?: string[]
  colSpan?: number
  rowSpan?: number
}
```

#### Returns

1. An Object matching the [Anatomy](#anatomy) containing `className` and a11y properties relevant to the Grid element including an unused class consisting of the naming pattern `pando-grid-item`.
2. A `style` property that adds dynamic styling based on the options provided.

## Behavior

There are no behavior patterns required for the Grid.

## Accessibility

The Pando Grid & GridItem APIs combined with semantic HTML allow the Grid & GridItems to be fully accessible and screen-readable.
