## Usage

### Basic 2 Column Layout

```jsx live
function TwoColLayout() {
  function getGridItemStyles(options) {
    const styles = getGridItemProps(options)
    return {
      style: {
        ...styles.style,
        backgroundColor: 'var(--ps-surface-strong)',
        borderRadius: '6px',
        height: '11rem',
        marginBottom: '1rem',
        padding: '1rem',
      },
    }
  }

  return (
    <div {...getGridProps({ cols: 12, gap: 32 })}>
      <div {...getGridItemStyles({ colSpan: 9 })}>main section</div>
      <div {...getGridItemStyles({ colSpan: 3 })}>sidebar</div>
    </div>
  )
}
```

## Reference

### `getGridProps`

```typescript
getGridProps(options?: GridOptions)
```

#### Parameters

- `options`: A Number value for properties cols and rows which determine the grid layout axises. A Number value for the property gap of `6 | 8 | 12 | 16 | 32` which determines the gap size for the grid.

#### Returns

An Object matching the [Anatomy](#anatomy) containing `className` and a11y properties relevant to the label element. Additionally, it returns a `style` property that adds dynamic styling based on the options provided.

### `getGridItemProps`

```typescript
getGridItemProps(options?: GridItemOptions)
```

#### Parameters

- `options`: A Number value for properties colSpan and rowSpan which determine the styling for the columns and rows span used for the grid item.

#### Returns

An Object matching the [Anatomy](#anatomy) containing `className` and a11y properties relevant to the label element. Additionally, it returns a `style` property that adds dynamic styling based on the options provided.
