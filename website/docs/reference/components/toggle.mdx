1. Wrapper
2. Label (optional)
3. Container
4. Input
5. Track
6. Thumb
7. Label (optional)

## Usage

### Basic Switch

```jsx live
function BasicSwitch() {
  const [checked, setChecked] = useState(false)
  const switchProps = getSwitchProps({
    checked,
    id: 'notfications',
    name: 'add_notifications',
  })

  function handleChange() {
    setChecked((prev) => !prev)
  }

  return (
    <FormControlProvider>
      <div {...switchProps.wrapper}>
        <Label htmlFor={switchProps.input.id}>Sign up for notifications</Label>
        <label {...switchProps.switchContainer}>
          <input {...switchProps.input} onClick={handleChange} />
          <span {...switchProps.switchTrack}>
            <span {...switchProps.switchThumb} />
          </span>
        </label>
      </div>
    </FormControlProvider>
  )
}
```

### Sizes

```jsx live
function SwitchSizes() {
  const [checked, setChecked] = useState(false)
  const switchProps = getSwitchProps({
    checked,
    id: 'notfications',
    name: 'add_notifications',
  })
  const smallSwitchProps = getSwitchProps({
    checked,
    id: 'notfications',
    name: 'add_notifications',
    size: 's',
  })

  function handleChange() {
    setChecked((prev) => !prev)
  }

  return (
    <form>
      <FormControlProvider>
        <div {...switchProps.wrapper}>
          <Label htmlFor={switchProps.input.id}>Default size</Label>
          <label {...switchProps.switchContainer}>
            <input {...switchProps.input} onClick={handleChange} />
            <span {...switchProps.switchTrack}>
              <span {...switchProps.switchThumb} />
            </span>
          </label>
        </div>
      </FormControlProvider>

      <FormControlProvider>
        <div {...smallSwitchProps.wrapper}>
          <Label htmlFor={smallSwitchProps.input.id}>Small size</Label>
          <label {...smallSwitchProps.switchContainer}>
            <input {...smallSwitchProps.input} onClick={handleChange} />
            <span {...smallSwitchProps.switchTrack}>
              <span {...smallSwitchProps.switchThumb} />
            </span>
          </label>
        </div>
      </FormControlProvider>
    </form>
  )
}
```

## Behavior

### Layout

**Do** use multiple Toggles in a vertical list.

**Don't** use multiple Toggles in a single row.

**Do** provide a Label with a Toggle.

**Don't** use a Toggle inline without a Label. Instead use a hidden Label.

### Usage

**Do** use a Toggle for giving users an way to opt-in to an optional feature.

**Don't** use a Toggle for collecting consent. Instead us a [Checkbox](./checkbox.mdx).

## API

### `Toggle: FC<ToggleProps>`

```typescript
type ToggleProps = HTMLAttributes<HTMLDivElement>
```

#### Parameters

1. Any props that can be passed to a `div` element.

### `ToggleButton: FC<ToggleButtonProps>`

```typescript
interface ToggleButtonProps extends InputHTMLAttributes<HTMLInputElement> {
  checked: boolean
  id: string
  name: string
  pandoSize?: 's' | 'm'
}
```

#### Parameters

1. Any props that can be passed to an `input` element.
2. `checked`: Whether the Toggle is checked or not.
3. `id`: The id of the Toggle.
4. `name`: The name of the Toggle.
5. `pandoSize`: The size of the Toggle.

### Size Mapping

```typescript title="Toggle size rules."
{
  s: 'Use in condensed areas where the default size is too large.',
  m: 'Should be used in most, if not all cases.'
}
```

## Accessibility

The Pando Toggle API combined with semantic HTML allow the Toggle to be fully accessible and screen-readable.
