## Import

```javascript
import { getTagProps } from '@pluralsight/headless-styles'
```

## Usage

### Tag Kinds

`getTagProps` comes with two kinds to change the visual style of a tag. You can use `weak` (default) and `strong`.

<BasicTag />
<LiveExampleTabs
  fullCode={<BasicTagFullPreview />}
  sandboxList={{
    chakra: 'basic-tag-chakra-q17lsd',
    joy: 'basic-tag-mui-xdhfm8',
    styled: 'basic-tag-css-in-js-sc-ykiwwt',
    svelte: 'ps-svelte-49dmmu',
  }}
>
  <BasicTagPreview />
</LiveExampleTabs>

### Tag Sizes

To change the size of a Tag, use the `size` option. The options are: `s`, `m` (default).

<TagSizes />
<LiveExampleTabs
  fullCode={<TagSizesFullPreview />}
  sandboxList={{
    chakra: 'basic-tag-chakra-q17lsd',

  }}

  <TagSizesPreview />
</LiveExampleTabs>

:::tip

We use the standard clothing size shorthand values for all `size` related options in any library we provide (i.e. xs-xxl).

:::

## API

| Name                    | Argument Type                          | Default                     | Description                      |
| ----------------------- | -------------------------------------- | --------------------------- | -------------------------------- |
| `getTagProps`           | **options**: [TagOptions](#tagoptions) | kind: 'weak'<br />size: 'm' | Get main tag styles.             |
| `getJSTagProps`         | **options**: [TagOptions](#tagoptions) | same as CSS API             | CSS-in-JS API                    |
| `getTagWithIconProps`   | **options**: [TagOptions](#tagoptions) | kind: 'weak'<br />size: 'm' | Get tag styles and icon options. |
| `getJSTagWithIconProps` | **options**: [TagOptions](#tagoptions) | same as CSS API             | CSS-in-JS API                    |

## Options

### TagOptions

This interface is available as an `import` for use.

```typescript
export interface TagOptions {
  kind?: 'weak' | 'strong'
  size?: 's' | 'm'
  tech?: 'svelte' | ''
}
```

```typescript title="Import example"
import type { TagOptions } from '@pluralsight/headless-styles/types'
```
