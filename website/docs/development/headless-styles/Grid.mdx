## Import

```javascript
import { getGridProps, getGridItemProps } from '@pluralsight/headless-styles'
```

- **getGridProps:** The main wrapper with `display: grid`
- **getGridItemProps:** A child of the grid to control the span and start positions within the grid.

## Usage

### Basic Two Column Layout

:::tip

Most designers use a standard 12 column layout. If you do not have access to a designer, we recommend using this value for the `cols` option.

:::

```jsx live
function TwoColLayout() {
  // ðŸ’¡ Normally you would make a Feature or Sidebar component.
  // This is only used to show what is happening in the example above.
  function getGridItemStyles(options) {
    const styles = getGridItemProps(options)
    return {
      style: {
        ...styles.style,
        backgroundColor: 'var(--ps-surface-strong)',
        borderRadius: '6px',
        height: '11rem',
        marginBottom: '1rem',
        padding: '1rem',
      },
    }
  }

  return (
    <div {...getGridProps({ cols: 12, gap: 32 })}>
      <div {...getGridItemStyles({ colSpan: 9 })}>main section</div>
      <div {...getGridItemStyles({ colSpan: 3 })}>sidebar</div>
    </div>
  )
}
```

## API

| Name                 | Argument Type                                    | Default                                       | Description               |
| -------------------- | ------------------------------------------------ | --------------------------------------------- | ------------------------- |
| `getGridProps`       | **options**: [GridOptions](#gridoptions)         | **cols:** 2<br />**gap:** 16<br />**rows:** 1 | Get grid container styles |
| `getJSGridProps`     | **options**: [GridOptions](#gridoptions)         | same as CSS API                               | CSS-in-JS API             |
| `getGridItemProps`   | **options**: [GridItemOptions](#griditemoptions) | **colSpan:** 1<br />**rowSpan:** null<br />   | Get grid item styles      |
| `getJSGridItemProps` | **options**: [GridItemOptions](#griditemoptions) | same as CSS API                               | CSS-in-JS API             |

## Options

### GridOptions

```typescript
export interface GridOptions {
  cols?: number
  gap?: 6 | 8 | 12 | 16 | 32
  rows?: number
  tech?: 'svelte' | ''
}
```

```typescript title="Available type export"
import type { GridOptions } from '@pluralsight/headless-styles/types'
```

### GridItemOptions

```typescript
export interface GridItemOptions {

  rowSpan?: number | null

}
```
