## Import

```javascript
import { getTabProps } from '@pluralsight/headless-styles'
```

:::tip

This helper only controls the styling. If you would also like a helper to control the content management and accessibility of Tabs, check out the React Utils [useTabs hooks](../react-utils/use-tabs.mdx)

:::

## Accessibility

### Relationship and State

The relationship and state of tabs and panels are expressed using ARIA roles and attributes.
This relationship can be demonstrated by the following property assignments:

```js
tab: {
  'aria-controls': panelId,
  'aria-selected': selectedTabId === tabId,
  id: tabId,
  tabIndex: focusedTabId === tabId ? 0 : -1,
},
tabPanel: {
  'aria-labelledby': tabId,
  'aria-hidden': selectedTabId !== tabId,
  'aria-expanded': selectedTabId === tabId,
  id: panelId,
  tabIndex: selectedTabId === tabId ? 0 : -1,
}

```

### Keyboard Navigation and Focus Management

[Keyboard navigation](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/tablist_role#keyboard_interactions)
for tabs involves using <kbd>arrow</kbd> keys to change focus and <kbd>enter</kbd>/<kbd>space</kbd> to activate the focused tab.

The preference is to use a
[roving tabindex](https://developer.mozilla.org/en-US/docs/Web/Accessibility/Keyboard-navigable_JavaScript_widgets#technique_1_roving_tabindex)
for managing focus and tab order.

- When a tab receives focus, set its `tabindex="0"` and `tabindex="-1"` for all others.
- Activating a tab also gives it focus.
- The active tabPanel will have `tabindex="0"` (independent of the focused tab).

## Hook Usage

To control the Tab active/display states, we provide a set of custom hooks in our [React Utils Library](../react-utils/use-tabs.mdx).

:::note

The useTabs custom hooks compliment the `getTabProps` helper to enhance your components further. Our packages are designed independantly so they can be combined together for more features when needed.

## API

| Name            | Argument Type                          | Default         | Description                   |
| --------------- | -------------------------------------- | --------------- | ----------------------------- |
| `getTabProps`   | **options**: [TabOptions](#taboptions) | size: 'm'       | Get tab styles for CSS usage. |
| `getJSTabProps` | **options**: [TabOptions](#taboptions) | same as CSS API | CSS-in-JS API                 |

## Options

### TabOptions

This interface is available as an `import` for use.

```typescript
export interface TabOptions {
  size?: 's' | 'm'
  tech?: 'svelte'
}
```

```typescript title="Import example"
import type { TabOptions } from '@pluralsight/headless-styles/types'
```

## Props

### TabsProps

```typescript
interface TabsProps {
  wrapper: {
    className: string
  }
  tabList: {
    className: string
    role: 'tablist'
  }
  tab: {
    'aria-selected': false
    className: string
    role: 'tab'
    tabIndex: -1
  }
  panelWrapper: {}
  tabPanel: {
    'aria-expanded': true
    'aria-hidden': false
    className: string
    role: 'tabpanel'
  }
}
```
