## Import

```javascript
import {
  getIconButtonProps,
  getDangerIconButtonProps,
} from '@pluralsight/headless-styles'
```

- **getIconButtonProps**: The styles for any button outside of status related
- **getDangerIconButtonProps**: The styles for danger status buttons

## Usage

### Icon Button kinds

`getIconButtonProps` comes with five kinds to change the visual style of an icon button. You can use `text` (default), `textWeak`, `weak`, `medium`, and `strong`.

### Icon Button styled links

If you want to use icon button styles for an `a` element, just omit the `type` property from the returned button object.

```jsx
const { button, iconOptions } = getIconButtonProps({ ariaLabel: 'Go back' })
const linkProps = {
  'aria-label' = button['aria-label'],
  className: button.className,
}

function BackLink() {
  return (
    <a href="#" {...linkProps}>
      <ArrowLeftIcon {...getIconProps(iconOptions)} />
    </a>
  )
}
```

## Extending

There are times you may need to make a slight adjustment to the icon button styles, which is easy to do since we are giving you an Object.

Here is an example of using `styled-components` to extend an icon button text color.

```jsx title=page/Login/components/CustomMenuButton.jsx
import styled from 'styled-components'
import {
  getJSIconButtonProps,
  getIconProps,
} from '@pluralsight/headless-styles'

const { button, iconOptions } = getJSIconButtonProps({
  ariaLabel: '',
  kind: 'medium',
})

const CustomButton = styled.button`
  ${button.cssProps}
  color: '#bfbfbf'
`

const CustomMenuIcon = styled(MenuIcon)`
  ${getIconProps(iconOptions).cssProps}
`

export default function CustomMenuButton(props) {
  return (
    <CustomButton {...props}>
      <CustomMenuIcon />
    </CustomButton>
  )
}
```

:::tip

When combining `headless-styles` with other libraries, please follow the guides in the example codesandboxes for the best experience. **Some libraries require special support**.

:::

## API

| Name                         | Argument Type                                                    | Default                                                             | Description                    |
| ---------------------------- | ---------------------------------------------------------------- | ------------------------------------------------------------------- | ------------------------------ |
| `getIconButtonProps`         | **options**: [IconButtonOptions](#iconbuttonoptions)             | **kind**: 'text', <br />**size**: 'm', <br />**variant**: 'default' | Get main icon button styles.   |
| `getDangerIconButtonProps`   | **options**: [DangerIconButtonOptions](#dangericonbuttonoptions) | **kind**: 'text', <br />**size**: 'm', <br />**variant**: 'default' | Get danger icon button styles. |
| `getJSIconButtonProps`       | **options**: [IconButtonOptions](#iconbuttonoptions)             | same as CSS API                                                     | CSS-in-JS API                  |
| `getJSDangerIconButtonProps` | **options**: [DangerIconButtonOptions](#dangericonbuttonoptions) | same as CSS API                                                     | CSS-in-JS API                  |

## Options

### IconButtonOptions

This interface is available as an `import` for use.

```typescript
interface IconButtonOptions {
  ariaLabel: string
  kind?: 'text' | 'textWeak' | 'weak' | 'medium' | 'strong'
  size?: 'xs' | 's' | 'm' | 'l'
  tech?: '' | 'svelte'
  variant?: 'default' | 'round'
}
```

```typescript title="Import example"
import type { IconButtonOptions } from '@pluralsight/headless-styles/types'
```

### DangerIconButtonOptions

```typescript
interface DangerIconButtonOptions extends IconButtonOptions {
  kind?: 'text' | 'medium' | 'strong'
}
```

## Props

### IconButtonProps

```typescript
interface IconButtonProps {
  button: {
    'aria-label': string
    className: string
    type: 'button'
  }
  iconOptions: {
    ariaHidden: true
    size: 's' | 'm' | 'l'
    tech: '' | 'svelte'
  }
}
```

```js title="Return value example"
{
  button: {
    'aria-label': 'Close dialog'
    className: `ps-icon-btn ${...internalStyles}`
    type: 'button'
  },
  iconOptions: {
    ariaHidden: true,
    size: 'm',
    tech: '',
  }
}
```

### JSIconButtonProps

```typescript
interface JSIconButtonProps {
  button: {
    'aria-label': string
    cssProps: TemplateLiteralStringArray | string
    styles: Record<CSSProps, string>
    type: ButtonType
  }
  iconOptions: {
    ariaHidden: true
    size: 'm'
    tech: ''
  }
}
```

```js title="Return value example"
{
  button: {

    cssProps: `

      ...

      backgroundColor: '#000'
      '&:hover': {
        ...
      }
    },
    type: 'button',
  },
  iconOptions: {
    ariaHidden: true,
    size: 'm',
    tech: '',
  }
}
```

## Guides

### Icon Button size guide

Icon Button sizes are designed to be square (1:1 aspect ratio). Here is an interface that gives the sizing guide.

```typescript title="Example interface - does not exist in types"
interface IconButtonSizeGuide {
  xs: {
    width: '1.5rem'
    height: '1.5rem'
  }
  s: {
    width: '2rem'
    height: '2rem'
  }
  m: {
    width: '2.5rem'
    height: '2.5rem'
  }
  l: {
    width: '3rem'
    height: '3rem'
  }
}
```

### Icon Button color guide

Icon Button colors are relative to the `background` style and have a direct correlation from the `kind` to the [`design-tokens`](../tokens/intro.md) package. Here is an interface that gives the color guide.

```typescript title="Example interface - does not exist in types"
interface IconButtonColorGuide {
  text: 'transparent'
  textWeak: 'transparent'
  weak: 'var(--ps-neutral-background)'
  medium: 'var(--ps-background)'
  strong: 'var(--ps-background-weak)'
}

interface DangerIconButtonColorGuide {
  text: 'transparent'
  medium: 'var(--ps-danger-background)'
  strong: 'var(--ps-danger-background-weak)'
}
```
