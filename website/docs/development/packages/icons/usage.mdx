# Usage (Web)

:::caution

This is **experimental** documentation for Pluralsight Design **icons** package.

:::

## Installation

To use icons in your application, install the npm package.

```bash npm2yarn
npm install @pluralsight/icons@alpha
```

## Icon colors

By default the icons are monochromatic and will inherit the `color` of their parent.
This ensures the icon color matches its context by default and facilitates
theming while still allowing for control over the icon's color directly.

## Icon size guide

We have defined four standard sizes for icons. This sizing should be applied using the [headless-styles](../headless-styles/Icon.mdx#icon-sizes) package.

```typescript title="Example interface - does not exist in types"
interface IconSizeGuide {
  s: {
    height: '1rem'
    width: '1rem'
  }
  m: {
    height: '1.25rem'
    width: '1.25rem'
  }
  l: {
    height: '1.5rem'
    width: '1.5rem'
  }
}
```

## Importing by framework

We export React components by default, which can be imported by name from the package root.

```javascript
import { MenuIcon } from '@pluralsight/icons'
```

We also export Svelte components.

```javascript
import { MenuIcon } from '@pluralsight/icons/svelte'
```

If needed, you can import SVGs directly. The approach will vary based on your framework and tooling.

```javascript title="Importing SVG in React with create-react-app"
import { ReactComponent as MenuArrowLeftIcon } from '@pluralsight/icons/svg/navigation/menu-arrow-left.svg'
```
