# Usage (Web)

:::caution

This is **experimental** documentation for Pluralsight Design **icons** package.

:::

## Installation

To use icons in your application, install the npm package.

```bash npm2yarn
npm install @pluralsight/icons@alpha
```

## Icon colors

By default the icons are monochromatic and will inherit the `color` of their parent.
This ensures the icon color matches its context by default and facilitates
theming while still allowing for control over the icon's color directly.

## Icon size guide

We have defined four standard sizes for icons. This sizing should be applied using the [headless-styles](../headless-styles/Icon.mdx#icon-sizes) package.

```typescript title="Example interface - does not exist in types"
interface IconSizeGuide {
  s: {
    height: '1rem'
    width: '1rem'
  }
  m: {
    height: '1.25rem'
    width: '1.25rem'
  }
  l: {
    height: '1.5rem'
    width: '1.5rem'
  }
}
```

## Importing by framework

Icons can be imported by name from the package root for React.

```javascript
import { MenuIcon } from '@pluralsight/icons'
```

If you are using a different, supported framework, append its name to the path.

```javascript
import { MenuIcon } from '@pluralsight/icons/svelte'
```

:::note

If your bundler does not support tree shaking, please see our notes on [minimizing bundle size](#minimizing-bundle-size)

:::

### Supported frameworks

We provide simple, ready-to-use components for the following frameworks:

- React
- Svelte

If you would like to request an additional format, please follow our [contribution process](../../discover-more/contributing.md#contributing).

## Importing SVG files (fallback)

:::danger

Directly importing the SVG files is meant only for unsupported frameworks and is not recommended.
This approach requires that you provide any tooling necessary for importing and using the SVG elements in your environment.

:::

```jsx title="Importing an icon as SVG file into an app created with create-react-app"
import { ReactComponent as ChatIcon } from '@pluralsight/icons/svg/communication/chat-message.svg'
import { getIconProps } from '@pluralsight/headless-styles'

function MyChatIcon(props) {
  return <ChatIcon {...getIconProps()} {...props} />
}
```
