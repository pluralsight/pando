The `getSwitchProps` helper is used as an alternative for the [Checkbox](./button) for choosing between enabled and disabled states.

## Import

```javascript
import { getSwitchProps } from '@pluralsight/headless-styles'
```

The Switch must always be accompanied by a label, and follows the same keyboard workflow as a `checkbox` (spacebar to toggle states).
