If you are using a bundler (like Webpack or Vite), you will need to add aliases for Pando paths:

```tsx
alias: {
  '@/styled-system/patterns': path.resolve(__dirname, './styled-system/patterns'),
  '@/styled-system/css': path.resolve(__dirname, './styled-system/css'),
  '@/styled-system/recipes': path.resolve(__dirname, './styled-system/recipes'),
}
```

:::tip[Live example]

See an example of this live in <TextLink href="https://github.com/pluralsight/pando/blob/main/docs/astro.config.mjs">the Pando docs configuration</TextLink>

:::
