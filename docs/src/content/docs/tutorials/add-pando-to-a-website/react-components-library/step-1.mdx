Panda CSS is the foundation to v1 and you will need to follow the setup instructions that compliment your team's project:

- PostCSS Plugin: <TextLink href='https://panda-css.com/docs/installation/postcss'>PostCSS</TextLink>
- Next.JS: <TextLink href='https://panda-css.com/docs/installation/nextjs'>Next.JS</TextLink>
- Vite: <TextLink href='https://panda-css.com/docs/installation/vite'>Using Vite</TextLink>
- Svelte: <TextLink href='https://panda-css.com/docs/installation/svelte'>Using Svelte</TextLink>

:::tip[Check in]

Let's do a sanity check to make sure that Panda CSS works.

A good way to check is to use the `css` function from Panda CSS in your project.

```tsx
import { css } from '../styled-system/css' // This path may differ depending on your project setup and where your `styled-system` directory generates

function TestPanda() {
  return (
    <div className={css({ backgroundColor: 'Chartreuse' })}>I am a test!</div>
  )
}
```

:::
