## Description

Pando's `<CicrularProgress />` is a visual component that is used to display the progress status for a task that takes a long time or consists of several steps within a page.

:::note[When to use]

Circular Progress should not be used for initial page loading. We recommend the use of <TextLink href="https://github.com/pluralsight/pando/blob/main/packages/react/src/components/Skeleton.tsx">Skeleton</TextLink> for initial Page Loading experiences.

:::

**Default**

```tsx

{
  ariaLabel: null,
  cycle:'determinate',
  displayValue: false,
  size: 'md',
  valuenow: 0,
  valuemin: 0,
  valuemax: 100,
```
