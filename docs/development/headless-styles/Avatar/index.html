Avatar

The Avatar is used to represent a user and is designed to display a profile image, initials, or fallback icon.

View source

Import‚Äã

import { getAvatarProps } from '@pluralsight/headless-styles'

Usage‚Äã

Avatar Kinds (fallback only)‚Äã

The getAvatarProps helper has two kinds to change the visual color of the avatar fallback that directly correlate to our design-tokens. You can use default and action.

note

When using Avatars, make sure there is a purpose behind the fallback colors you are choosing. Follow our guide to learn how to display avatars for the best user experience.

daaa
const avatar = getAvatarProps({
  label: 'Pluralsight avatar'
  sentiment: 'default',
})

<span {...avatar.wrapper}>
  <span {...avatar.label}>{avatar.label.value}</span>
</span>

Avatar Sizes‚Äã

To change the size of an Avatar, use the size option. The options are: xs, s, m (default), l, and xl.

xssmlxl
const avatar = getAvatarProps({
  label: 'extra large avatar',
  size: 'xl'
})

<span {...avatar.wrapper}>
  <span {...avatar.label}>xl</span>
</span>
tip

We use the standard clothing size shorthand values for all size related options in any library we provide (i.e. xs-xxl).

Avatar Content‚Äã

The DS Avatars have three different states to display to the user. Each state has a different purpose for usage and should follow the guides below.

Image (preferred)‚Äã

A profile image should be used for all Avatars no matter the context (i.e. person, platform, etc.). This will give the best visual experience for the user and should be used over all other options.

const avatarProps = getAvatarProps({
  label: 'Harry Potter',
  src: 'http://harrypotter.com/avatar'
})

<span {...avatarProps.wrapper}>
  <img {...avatarProps.image} />
</span>

Initials (fallback)‚Äã

In the absence of an image, the avatar's initials (up to 2 characters) should be displayed. This is not ideal but allows the user to still quickly put the dots together of who the avatar is meant to represent.

const avatarProps = getAvatarProps({
  label: 'Harry Potter',
  src: ''
})

<span {...avatarProps.wrapper}>
  <div {...avatarProps.label}>{avatarProps.label.value}</div>
</span>

Icon (last resort)‚Äã

If neither the image nor initials are available, display an icon using the iconOptions from the AvatarProps Object to represent a person.

danger

Although this option exists, it is not recommended to be used as it provides a vague user experience. If you have to use this option, you need to update your API to require a name for each user.

const avatarProps = getAvatarProps({
  label: '',
  src: ''
})

<span {...avatarProps.wrapper}>
  <PersonIcon {...getIconProps(avatarProps.iconOptions)} />
</span>

PreloadImage Hook‚Äã

For the best user-experience overall, combine this helper with our Preloaded Image Hook, React Suspense component, and the Skeleton API.

You can even add a custom ErrorBoundary that will display the initials or icon avatar if the links crash. Using this approach will provide the best user experience while allowing React to manage the image state. ‚ö°

VIEW LIVE EXAMPLE üëÄ

function Fallback() {
  return <div {...getSkeletonProps({ kind: 'circle' })} />
}

function Image(props) {
  const { imgData, ...avatarProps } = props
  const avatarProps = getAvatarProps(avatarProps)
  const img = imgData.read()

  return (
    <span {...avatarProps.wrapper}>
      <img {...avatarProps.image} {...img} />
    </span>
  )
}

function Avatar() {
  const resource = usePreloadedImg({
    alt: 'random image',
    src: 'https://source.unsplash.com/random/?face&fit=facearea&facepad=2&w=256&h=256&q=80',
  })

  return (
    <AvatarErrorBoundary>
      <div>
        {resource && (
          <Suspense fallback={<Fallback />}>
            <Image alt="test image" imgData={resource.img} />
          </Suspense>
        )}
        {!resource && null}
      </div>
    </AvatarErrorBoundary>
  )
}

Options‚Äã

AvatarOptions‚Äã

This interface is available as an import for use.

export interface AvatarOptions {
  label: string
  sentiment?: 'default' | 'strong'
  size?: 'xs' | 's' | 'm' | 'l' | 'xl'
  src: string
  tech?: 'svelte' | ''
}
Import example
import type { AvatarOptions } from '@pluralsight/headless-styles/types'

Props‚Äã

AvatarProps‚Äã

interface AvatarProps {
  iconOptions: {
    ariaHidden: true
    ariaLabel: string
    customSize: string
    tech: string
  }
  wrapper: {
    className: string
  }
  image: {
    alt: string
    className: string
    src: string
  }
  label: {
    'aria-label': string
    className: string
    value: string
  }
}

JSAvatarProps‚Äã

interface JSAvatarProps {
  iconOptions: {
    ariaHidden: true
    ariaLabel: string
    customSize: string
    tech: string
  }
  wrapper: {
    cssProps: TemplateLiteralStringArray | string
    styles: Record<CSSProps, string>
  }
  image: {
    a11yProps: {
      alt: string
      src: string
    }
    cssProps: TemplateLiteralStringArray | string
    styles: Record<CSSProps, string>
  }
  label: {
    a11yProps: {
      'aria-label': string
      value: string
    }
    cssProps: TemplateLiteralStringArray | string
    styles: Record<CSSProps, string>
  }
}
